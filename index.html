Harlem Coding Kids





At Harlem Coding Kids, children from ages 9-12 can learn to code for free!


It’s a 10-week group class of 3-5 children that meets online once a week. The class is conducted virtually—so you can learn from anywhere! At Harlem Coding Kids, your children can learn to code while making friends and having a fun, challenging, and useful experience.

WHY CODING?


Coding is an important skill and tool that your kids will be able to carry with them for the rest of their lives. Learning to code is not only fun but important because the world is becoming a tech-based place. Many people learn to code at a young age, and this knowledge will be a valuable tool in adulthood. While it may seem complicated, it’s an accessible skill for all ages.

Coding contributes to almost all aspects of modern life. Coding is at the heart of your cell phone and your apps, cars, and even features of everyday life that may not initially seem to be coding-related, like the traffic lights that guide us through the city.


With lessons from Harlem Coding Kids, your children will be able to have conversations about code as well as understand the world in a way that they wouldn’t otherwise!

WHY PYTHON?


Coding takes place in different “languages,” such as Python, Java, and C++. This class will focus on the basics of Python, one of the most commonly used and easily readable coding languages. It’s widely considered the best language with which to begin your coding journey.

Python allows you to do many things. It can be used to create programs such as simple games and translators for different languages, study tools, or secret codes creator and decoder for private communication. It can also be used to analyze data. Eventually, you can complete complex mathematical functions and create more sophisticated programs. It is helpful in many fields, from finance to medicine.


After mastering Python, you can go on to learn other coding languages with a solid foundation.
